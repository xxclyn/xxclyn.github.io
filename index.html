web前端知识点汇总

六种原始类型

boolean
null
undefined
number
string
symbol

原始类型存储的都是值，没有方法可以调用，某些情况下，原始类型会被强转为包装对象，从而可以调用toString()之类的函数

number类型是浮点类型，所以0.1+0.2!==0.3

symbol类型表示独一无二的值，通常用来作为属性名使用

对象(Object)类型

原始类型存储的是值，对象类型存储的是地址（指针），所以修改对象的属性会影响其他指向该地址的对象

typeof可以用来判断除了null以外的所有原始类型，tyoeof null会返回’object’，这是js的一个bug

typeof一个对象，对象是函数会返回’function’，其余都返回’Object’

instanceof是通过原型链来进行判断的

类型转换

js中类型转换有3种情况，转换为布尔值，转换为数字，转换为字符串

条件判断时，undefined，null，false，NaN，””，0，-0，以上会被转换成false，其他都是true，包括对象

加法运算符
如果其中一方是字符串，会把另一方也转换为字符串
如果一方不是字符串或者数字，就会把它转换为字符串或者数字

不是加法运算符，只要一方是数字，另一方就会被转换为数字

this

一般情况下，谁调用了函数，谁就是this

对于new方式来说，this永远被绑定在他的实例上

箭头函数的this取决于包裹箭头函数的第一个普通函数的this

bind方法可以改变上下文环境

闭包

函数内部嵌套函数，内部的函数可以访问到外部函数中的变量，就是闭包

闭包存在的意义就是可以间接访问到函数内部的变量

更多关于闭包的内容可以看这篇文章(https://github.com/dwqs/blog/issues/18)

深浅拷贝

拷贝就是复制对象并让他互不影响

浅拷贝

Object.assign()
展开运算符…

深拷贝

JSON.parse(JSON.stringfy(Object))
lodash深拷贝函数

原型

对象的constructor属性指向其构造函数

对象的proto属性指向原型，即构造函数的prototype

prototype是函数特有的属性，他是一个对象
